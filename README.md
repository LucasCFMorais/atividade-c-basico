## Exercícios

1. Impressão de números de 1 até N
2. Cálculo de fatorial
3. Soma de números em array
4. Conversão de temperatura
5. Contagem de caracteres em string
6. Verificação de número primo

## Como compilar

É necessário ter o GCC instalado.

Exemplo de compilação:

gcc 1_numeros_1_ate_N.c -o numeros

## Como executar

Linux/Mac:

./numeros

Windows:

numeros.exe
